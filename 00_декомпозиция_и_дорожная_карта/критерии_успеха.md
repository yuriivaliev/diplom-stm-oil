# Критерии успеха диплома (черновик)

Согласованы этапы 0–7; измеримые критерии — для самопроверки перед защитой.

1. **Рынок:** таблица/график долей ключевых брендов **2022–2026** с реестром источников и явными допущениями (в т.ч. федеральный срез vs европейская часть).
2. **Продукт:** разрез спроса по **вязкостям** (SAE) с ограничениями данных и импликациями для линейки СТМ.
3. **Конкуренты:** сравнительная матрица **упаковки/этикеток** (PGMM + ODSA) по согласованному списку брендов.
4. **Бренд:** **2–3** кандидата названия с DDx/ODSA и отдельной проверкой рисков товарного знака (вне LLM).
5. **Дизайн:** **3–5** эскизов упаковки СТМ, согласованных с выводами PGMM и названием.
6. **Синтез:** единое **позиционирование СТМ**, выводы, глава методологии, блок **нормативки РФ/ЕАЭС** — не набор разрозненных отчётов LLM.
7. **Методология:** для каждого крупного артефакта — след протокола аналитика (таксономия → промпт → исполнение → Red/Blue).
