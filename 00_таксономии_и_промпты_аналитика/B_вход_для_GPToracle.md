## Блок для копирования в GPToracle

```
ЗАДАЧА МЕТА-ПРОМПТА
Собери один самодостаточный промпт для Deep Research по артефакту B дипломной работы:
динамика спроса по SAE (синтетика + полусинтетика) на рынке легковых автомасел
в ЕВРОПЕЙСКОЙ ЧАСТИ РФ (ЦФО + СЗФО + ЮФО), 2022–2026.

КОНТЕКСТ (двойная рамка)
- СТМ: линейка синт+полусинт; выбор SKU (10W-40, 5W-40, 5W-30, 0W-20 + опционально 0W-16, 5W-20, 0W-30).
- Диплом ДПО МГИМО: методология, ≥8 URL, таблицы, ограничения.
- DR-A (доли брендов) — ОТДЕЛЬНО; в B максимум 1 абзац отсылки, БЕЗ таблиц брендов.
- План 27.05.2026; минералка ИСКЛЮЧЕНА полностью.

ГЕОГРАФИЯ
- Ядро: ЦФО, СЗФО, ЮФО.
- Нет SAE по ФО → федеральный прокси + red flag «федеральный прокси».
- Качественно: парк, китайские марки в европ. части (Autostat).

ПЕРИОД
- 2022–2026; для 2026 искать промежуточные итоги (11М2025, H1, 12 мес. Nielsen).
- Нет данных → н/д; БЕЗ интерполяции.

PRIOR (гипотезы, перепроверить)
- 5W-30 ~35%, 5W-40 ~32% (старые оценки)
- 0W-20 ускоренный рост (+16% ₽ в 2023 — перепроверить)
- NL-01: рост 5W-30, 5W-40, 0W-20 (Kolesa/Nielsen)
- НЕ переносить: минералка, 15W-40 mineral, доли брендов

ДОМЕНЫ MECE: B0–B11 (см. B_домены_контекст_роли.md)

КРИТЕРИИ УСПЕХА
1) Таблица SAE × год/окно × доля % (объём; ₽ отдельно)
2) Обязательные SAE: 10W-40, 5W-40, 5W-30, 0W-20
3) GATE сумма ≈100% ±2 п.п.
4) Таблица 0: факт/промежуточный/н/д
5) Подраздел 0W-20 + китайский парк
6) 3–5 SKU СТМ с привязкой к таблице
7) ≥8 источников URL

ОГРАНИЧЕНИЯ
- Русский язык
- Не минералка; 10W-40 только синт/полусинт в базе
- Не смешивать розницу и total
- Не юридические выводы по маркировке

ФОРМАТ DR: 12 разделов markdown (см. B_таксономия 1.y)

РОЛИ: технолог СМ, категорийный менеджер, методолог-аудитор

ЗАПРОС К ORACLE
Сгенерируй финальный промпт DR:
- самораспаковывающийся;
- пошаговый алгоритм + приоритет источников (Nielsen, Aftermarket-DATA, GfK, Autostat);
- red flags (минералка в таблице, прокси без пометки, интерполяция);
- патчи BP1–BP12 (аналог P1–P12 для SAE).
```
