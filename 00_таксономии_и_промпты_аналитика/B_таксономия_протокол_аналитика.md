## 1.j — Задача (job)

Сформировать **мета-промпт для Deep Research** и **исполнительский промпт**, которые дадут **динамику спроса по SAE** (синтетика + полусинтетика, легковой + LCV) в **европейской части Российской Федерации** за **2022–2026** (с **промежуточными итогами** для 2026) и **3–5 импликаций для линейки СТМ**.

**Не входит в B:** доли брендов (→ DR-A); **минеральные масла** полностью; PGMM, название, визуал (→ C–E).

**Связь с A:** максимум 1 абзац отсылки; **без таблиц брендов**.

---

## 1.x — Критерии успеха (измеримые)

1. **География:** европейская часть = **ЦФО + СЗФО + ЮФО**; при отсутствии SAE по ФО — **федеральный прокси** + 🚩 + качественные драйверы (парк, климат).
2. Таблица: **SAE × (2022, 2023, 2024, 2025, 2026 или промежуточное окно) × доля %** (объём приоритет; ₽ — отдельно).
3. **Обязательные SAE:** **10W-40, 5W-40, 5W-30, 0W-20**; акцент на **0W-20** и **китайский парк**.
4. **Опционально:** 0W-16, 5W-20, 0W-30 — при устойчивых источниках.
5. **GATE:** сумма долей SAE (синт+полусинт) ≈ **100% ±2 п.п.** по году/окну.
6. **Таблица 0:** год/окно | факт / промежуточный / оценка / н/д | география | комментарий.
7. Реестр **≥8** источников с URL; tier; red flags.
8. Сверка prior DR (вязкости); **импликации СТМ** (3–5 SKU) с привязкой к таблице.
9. **2026:** искать 11М2025, H1/9M2025, 12 мес. Nielsen; иначе **н/д** — **без интерполяции**.

## 1.t — Алгоритм (для исполнителя DR)

1. Зафиксировать **рамку**: синт+полусинт, без минералки; европ. часть = ЦФО+СЗФО+ЮФО.
2. Собрать **SAE-доли** по годам 2022–2026 (Nielsen/Kolesa, Aftermarket-DATA, GfK/A-KT).
3. Для **2026** — промежуточные окна; иначе **н/д**.
4. Нормализовать строки: **0W-20, 5W-30, 5W-40, 10W-40** + опциональные + «прочие синт+полусинт».
5. **GATE** ±2 п.п. перед расчётом п.п.
6. Подраздел **0W-20**: китайский парк (Autostat, регистрации), OEM, e-com как индикатор.
7. **Европ. часть:** парк/продажи по ФО; при отсутствии SAE по ФО — федеральный прокси + 🚩.
8. Каналы — **качественно** + доли канала только при методе объёма/₽.
9. Шоки 2022, маркировка 2025 — кратко, без юр. выводов.
10. **Импликации СТМ** — 3–5 SKU из таблиц (не из брендов A).
11. **Реестр URL**; сверка prior; 1 абзац «контекст DR-A».
