## 1.j — Задача (job)

Сформировать **мета-промпт для Deep Research** и затем **исполнительский промпт**, которые дадут **сопоставимую динамику долей рынка** ключевых брендов моторных масел (синтетика + полусинтетика, легковой сегмент + LCV) в **РФ** за **2022–2026** с оговорками по **европейской части** и импликациями для запуска **СТМ**.

**Не входит в A:** спрос по вязкостям (SAE), PGMM, название, визуал — отдельные артефакты B–E.

---

## 1.x — Критерии успеха (измеримые)

1. Таблица: **бренд × год (2022–2026) × доля % (объём)** + при наличии **доля % (выручка)**.
2. Минимум **8 целевых брендов** с данными хотя бы за 3 года: LUKOIL, Gazpromneft, Rosneft, ZIC, Shell, Mobil, Castrol, Total (+ Elf при наличии источника).
3. Строка **«прочие / нераспределённый остаток»** и при возможности **СТМ торговых сетей**.
4. Блок **методология**: определение рынка, канал (розница/total), источник, дата доступа.
5. Блок **ограничения**: где только федеральный срез; где оценка/прогноз 2026.
6. Реестр **≥10** проверяемых источников (отчёты, Nielsen/GfK/отраслевые, регуляторика).
7. Раздел **импликации для СТМ** (3–5 буллетов): окно входа, конкуренты, каналы.
8. Prior DR (PDF) использован как **черновик гипотез**, все цифры **перепроверены** в новом DR.

## 1.t — Алгоритм (для исполнителя DR)

1. Зафиксировать **определение рынка** (1 абзац).
2. Собрать **объём/CAGR** по годам 2022–2026 (федерально).
3. По каждому **целевому бренду** — доля % по объёму по годам; при расхождении источников — две колонки + комментарий.
4. Свести **топ-10 + прочие**; проверить сумму ≈100% (допуск ±2 п.п. с пояснением).
5. Отдельно: **Shell, Mobil, Castrol, Total, Elf** — траектория после 2022.
6. Каналы: DIY / СТО / дилер / e-com — **структура**, не дублировать DR B.
7. Подраздел **европейская часть**: ЦФО, СЗФО, Юг — что известно; иначе — «нет публичного среза».
8. **СТМ сетей** — качественно + оценка доли при источнике.
9. **Импликации для СТМ** — только из таблиц, без маркетинговой фантазии.
10. **Реестр источников** — URL, дата, тип (первичный/вторичный).
