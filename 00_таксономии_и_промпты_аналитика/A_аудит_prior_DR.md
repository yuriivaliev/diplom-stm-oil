## Критические пробелы нового DR (обязательно в промпте)

1. **Временной ряд долей по брендам: 2022, 2023, 2024, 2025, 2026** (оценка/прогноз с пометкой).
2. **Единица сопоставимости:** % по объёму (л) — основная; % по выручке (₽) — дополнительная колонка.
3. **Методология доли:** розница / total market / синтетика+полусинтетика — явно в шапке таблицы.
4. **География:** РФ (федерально) + **отдельный подраздел** «европейская часть РФ» (ЦФО + СЗФО + Юг как минимум; если данных нет — явная оговорка).
5. **СТМ сетей** (Ашан, Лента, Fix Price, прочие private label) — отдельная строка или сноска.
6. **Не смешивать** с вязкостями (SAE 0W-20 и т.д.) — только отсылка «см. DR B».

## Риски повторного использования prior DR без перепроверки

- Цифры **одного года** (2023) могут быть приняты за динамику.
- Смешение **всего рынка масел** (300–480 млн л) и **легкового розницы** (278 млн л).
- **ZIC «2-е место»** в narrative vs таблица (5,5%) — нужна сверка источника.
- Footnote-номера в PDF — не источники; нужны URL/отчёты.
