DEEP RESEARCH PROMPT

ТЕМА И ЦЕЛЬ ИССЛЕДОВАНИЯ

Подготовь исследовательский артефакт B для дипломной работы ДПО МГИМО на тему:

«Динамика спроса по классам вязкости SAE (только синтетические и полусинтетические моторные масла для легковых автомобилей) в европейской части Российской Федерации (ЦФО + СЗФО + ЮФО) за период 2022–2026 гг. с выводами для формирования линейки СТМ».

Работай как объединённая команда экспертов:
1. Технолог по смазочным материалам.
2. Категорийный менеджер рынка автомасел.
3. Методолог-аудитор академических исследований.

КЛЮЧЕВАЯ ЗАДАЧА

Определить и документировать изменение структуры спроса по SAE-классам синтетических и полусинтетических моторных масел для легковых автомобилей в 2022–2026 гг., выявить основные тренды и сформировать рекомендации по SKU для собственной торговой марки.

ИССЛЕДОВАТЕЛЬСКИЕ ВОПРОСЫ

1. Как менялись доли SAE-классов по объёму продаж и отдельно по денежному выражению?
2. Какие SAE демонстрировали рост, стагнацию или снижение?
3. Насколько вырос сегмент маловязких масел (прежде всего 0W-20)?
4. Как изменение структуры автомобильного парка и рост китайских брендов влияет на спрос по SAE?
5. Какие 3–5 SKU являются приоритетными для запуска линейки СТМ?

ОБЛАСТЬ ИССЛЕДОВАНИЯ

География:
- ЦФО.
- СЗФО.
- ЮФО.

Если SAE-данные отсутствуют на уровне федеральных округов:
- использовать только федеральные данные РФ как прокси;
- обязательно маркировать каждый такой показатель пометкой:
  «Федеральный прокси — отсутствуют данные по ЦФО/СЗФО/ЮФО».

Период:
- 2022.
- 2023.
- 2024.
- 2025.
- 2026.

Для 2026 разрешено использовать:
- H1 2026;
- 11М 2025;
- rolling 12 months;
- последние доступные периоды Nielsen/GfK/Aftermarket-DATA.

Если данных нет:
- указывать «н/д»;
- запрещено выполнять интерполяцию;
- запрещено рассчитывать доли методом экстраполяции.

- запрещены таблицы долей брендов;
- запрещён перенос брендовых выводов из артефакта A.
