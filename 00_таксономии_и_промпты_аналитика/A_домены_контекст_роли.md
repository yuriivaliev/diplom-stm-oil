# DR-A: домены (MECE), контекст, роли

Домены — **взаимоисключающие блоки** постановки для Oracle и DR. Каждый домен = отдельный подзапрос внутри одного DR (не отдельный чат).

## Контекстный пакет (вставлять в Oracle / DR)

### Бизнес-контекст (двойная рамка)
- **Проект запуска СТМ** автомоторных масёл в РФ (фокус **европейская часть** для GTM) — основная прикладная цель.
- **Диплом ДПО МГИМО** — оформление: методология, реестр источников, проверяемые таблицы, след протокола аналитика.
- Конкурентное поле: массовый и средний сегмент доминируют; премиум сжат после 2022.
- Решения покупателя на полке и в СТО часто принимает **мастер/продавец** → важны каналы b2b2c.

### Исторический контекст (из prior DR — гипотезы для проверки)
- 2023 розница легковой: **~278 млн л**, **~196 млрд ₽**.
- Топ-2023 (объём %): LUKOIL 18,1; Gazpromneft 10,5; Rosneft 8,5; ZIC 5,5; Shell 4,5; Sintec 4,2; Kixx 3,1; LUXE 3,0; Mobil 2,2; G-Energy ~2,0.
- Shell+Mobil суммарно **<10%** после ухода официальных поставок.
- ЦФО ~**25%** объёма (оценка); европейская часть — высокая доступность импорта.

### Целевые бренды (обязательный охват)
`LUKOIL`, `Gazpromneft`, `Rosneft`, `ZIC`, `Shell`, `Mobil`, `Castrol`, `Total`, `Elf` (если данные), `SINTEC`, `Rolf`, `Kixx` — второй эшелон по наличию данных.

### Иерархия источников (приоритет)
1. Отраслевые аудиты: **Nielsen**, **GfK**, **Aftermarket-DATA**
2. Отчёты производителей / дистрибьюторов (публичные)
3. **Росстат**, отраслевые ассоциации
4. Качественные СМИ/аналитика — только с перекрёстной сверкой
5. Prior DR (PDF) — **не источник**, только гипотезы

## Виртуальные роли для DR (встроить в промпт исполнителя)

```
Ты — команда из трёх ролей (явно маркируй, кто отвечает в каждом разделе):

1. **Ведущий аналитик рынка смазочных материалов (РФ)**  
   — определения, объёмы, доли, CAGR, таблицы.

2. **Категорийный менеджер автодилера/ритейла**  
   — каналы, СТМ сетей, поведение СТО/DIY, импликации для СТМ.

3. **Методолог-аудитор данных**  
   — проверка источников, пометки «оценка/прогноз», конфликты цифр, 
     сравнение с prior DR (подтверждено/опровергнуто).
```

**Системный стиль (опционально в отдельном system):** нейтральность, MECE, без рекламных формулировок — по `system_instructions_neuromarketing.md` (сжато, не весь файл).
