# Вход для GPToracle (прогон A)

**Ссылка:** [GPToracle — The AI Research Meta Prompter](https://chatgpt.com/g/g-67cf7a6be27c8191835abbb61a8f9bed-gptoracle-the-ai-research-meta-prompter)

**Инструкция:** вставьте блок ниже **целиком** в Oracle. Приложите файл `A_таксономия_протокол_аналитика.md` или краткое резюме prior DR (таблица 2023). Результат Oracle сохраните как `A_промпт_DR_исполнитель_v2_oracle.md`.

## Блок для копирования в GPToracle

```
ЗАДАЧА МЕТА-ПРОМПТА
Собери один самодостаточный промпт для Deep Research (ChatGPT DR / Gemini Deep Research) 
по артефакту A дипломной работы: динамика долей рынка моторных масел в РФ, 2022–2026.

КОНТЕКСТ ПРОЕКТА (двойная рамка — указать обе)
- **Проект запуска СТМ** (собственная торговая марка автомоторных масел, синтетика + полусинтетика) — 
  прикладная цель: доли, конкуренты, каналы, импликации для GTM.
- **Диплом ДПО МГИМО** — академическое оформление: методология, ≥10 источников, 
  таблицы, ограничения данных, без маркетинговой «воды».
- География выводов: РФ; для GTM — европейская часть (ЦФО, СЗФО, Юг). 
  Если нет региональных данных — явная оговорка «федеральный прокси».
- План согласован с руководителем 27.05.2026. Все DR декомпозировать; 
  этот промпт — ТОЛЬКО доли брендов, БЕЗ вязкостей SAE.

ПОЛУЧАТЕЛЬ РЕЗУЛЬТАТА DR
Научный руководитель + автор диплома. Нужны проверяемые таблицы и реестр источников.

PRIOR RESEARCH (гипотезы, перепроверить)
Ранее выполнен DR без Oracle (PDF, 13 стр., легковой РФ, синт/полусинт):
- 2023 розница ~278 млн л, ~196 млрд ₽
- Топ-2023 объём %: LUKOIL 18,1; Gazpromneft 10,5; Rosneft 8,5; ZIC 5,5; 
  Shell 4,5; Sintec 4,2; Kixx 3,1; LUXE 3,0; Mobil 2,2; G-Energy ~2,0
- После 2022: уход официальных западных брендов, параллельный импорт, 
  рост российских и азиатских марок
Пробелы prior DR: нет ряда 2022–2026 по брендам; слабо Castrol/Elf; нет СТМ сетей; 
нет отдельного среза «европейская часть».

ОБЯЗАТЕЛЬНЫЕ ДОМЕНЫ (MECE, 15 блоков)
D0 рамка рынка | D1 ёмкость/CAGR | D2 доли по объёму (ядро) | D3 доли в ₽ |
D4 импорт vs локал | D5 Shell/Mobil/Castrol/Total/Elf post-2022 |
D6 LUKOIL/Gazpromneft/Rosneft/SINTEC/Rolf | D7 ZIC/Kixx/азиатские |
D8 каналы (DIY/СТО/дилер/e-com) | D9 ценовые ярусы |
D10 европейская часть РФ | D11 СТМ/private label сетей |
D12 шоки (2022, маркировка 2025) | D13 методология/источники | D14 импликации для СТМ

ЦЕЛЕВЫЕ БРЕНДЫ (минимум)
LUKOIL, Gazpromneft, Rosneft, ZIC, Shell, Mobil, Castrol, Total, Elf (если есть данные),
SINTEC, Rolf, Kixx — плюс строка «прочие».

КРИТЕРИИ УСПЕХА
1) Таблица бренд × (2022,2023,2024,2025,2026) × доля % объём
2) При наличии — доля % выручка
3) Методология в шапке (розница vs total; синт+полусинт; легковой+LCV)
4) ≥10 источников с URL и датой
5) Раздел: prior DR — что подтверждено/опровергнуто
6) Не выдумывать цифры; оценки помечать
7) НЕ включать разрез по вязкостям SAE (отдельное исследование B)

ОГРАНИЧЕНИЯ
- Русский язык отчёта
- Не смешивать в одной таблице «весь рынок масел» и «легковой розницы»
- Не юридические заключения по нормативке — только факты влияния на рынок

ФОРМАТ ВЫХОДА DR
Структура markdown из 12 разделов (резюме → определение → объём → таблицы долей → 
динамика п.п. → каналы → европейская часть → СТМ → импликации СТМ → источники → ограничения → 
сверка с prior DR).

РОЛИ В ПРОМПТЕ ИСПОЛНИТЕЛЯ
Встроить три роли: (1) аналитик рынка СМ РФ, (2) категорийный менеджер ритейл/СТО, 
(3) методолог-аудитор данных.

ЗАПРОС К ORACLE
Сгенерируй финальный промпт для Deep Research, который:
- самораспаковывающийся (исполнитель не задаёт уточняющих вопросов);
- содержит пошаговый алгоритм сбора;
- задаёт приоритет источников (Nielsen, GfK, Aftermarket-DATA, отраслевые отчёты);
- включает инструкцию Red-флагов (противоречивые цифры, отсутствие года, путаница объём/₽).
```
