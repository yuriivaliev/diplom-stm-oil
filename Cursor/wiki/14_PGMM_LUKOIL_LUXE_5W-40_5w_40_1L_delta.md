## 2. Delta по блокам

### 2.1. C — Носитель (Carrier)

| Параметр | 4L (база) | 1L (delta) | Δ |
|----------|-----------|------------|---|
| **Геометрия** | Кастомная асимметричная канистра; левый щит этикетки сращён с корпусом (Gestalt-lock) | Узкая вертикальная «бутылка»; левая грань — плавная кривая, правая — вертикаль с эрго-впадиной | **Смена морфологии:** сосуд замены → модуль долива |
| **Ручка** | Интегрированная ручка + тактильные «пупырышки» | **Нет отдельной ручки**; треугольная зона с рельефными точками на правом плече | **Новый affordance:** одноручный захват/налив |
| **Материал** | Металлизированный золотой PE + выборочный лак | Золотой/бронзовый сатиновый PE; этикетка в **вдавленном щите** | Константа кодировки «золото = ценность»; **↑ интеграция** label-in-recess |
| **Пропорции H:W** | Широкий, устойчивый силуэт 4L | Высокий, узкий, «towering» | **DoF сжат по ширине, растянут по высоте** |
| **Крышка** | Красный акцент (база) | Двухтоновая: красная рифлёная + золотое кольцо горловины | ✅ сохранён красный якорь; **↑ premium finish** |
| **Стабильность** | Широкая база, низкий CG | Узкая база, высокий CG | **System Dynamics — Limit:** риск опрокидывания при плотной выкладке |

**Вывод (C):** Носитель 1L — не уменьшенная 4L-канистра, а **перекодировка физики**: с «бронированного сосуда полной замены» на «портативный золотой модуль долива». Тактильный код сохранён, но **локализован** на плечо (locc_001). Золотой монолит (MCON_03) на 1L **визуально усилен** за счёт большей доли «голого» корпуса вокруг щита.

### 2.2. K — Композиция (Layout)

| Параметр | 4L (база) | 1L (delta) | Δ |
|----------|-----------|------------|---|
| **Стратегия макета** | `dynamic_split_screen`; `K_PATTERN_SHIELD_AND_CORE` + `K_PATTERN_STRATIFIED_TELEMETRY` | Вертикальная **рескейл-копия** щитового макета на узкий носитель | **K_PATTERN_VERTICAL_COMPRESS** (locc_006) |
| **Negative space** | Чёрная «бездна» (MCON_08) + стратифицированные полосы | Щит занимает **бóльшую долю** фронтальной площади; «воздуха» между элементами меньше | MCON_08 **сжат**, не инвертирован (в отличие от Mobil) |
| **Иерархия** | Бренд → LUXE → спецификации | Идентичная последовательность; **1L** вместо 4L в футере | Константа порядка |
| **Читаемость SAE 5W-40** | Жёсткая изоляция внизу (MCON_02, MCON_09) | Сохранена; белый жирный шрифт на тёмной полосе; **относительный размер ↑** к ширине | **Усилена** на доливном формате |
| **Читаемость объёма** | «4L» — якорь полной замены | «1L» — правая секция футера; **мельче**, чем 5W-40 | **Семантический сдвиг:** maintenance vs full change |
| **Плотность** | Гипер-маскулинный авторитарный стиль | Все ключевые блоки сохранены; LUXE ≈ **40%** вертикали этикетки | **↑ cognitive density** без потери primary layer |
| **Новые K-элементы** | — | Иконка легкового авто в золотой полосе «УНИВЕРСАЛЬНОЕ МАСЛО» | ➕ locc_008 |
| **Выпавшие K-элементы** | ActiPure, benefit-блоки (база) | **Не видны** на фронтали 1L | locc_007_LACUNA |

**POV — Information Integrity:**  
- *Redundancy:* дублирование «SYNTHETIC» (латиница) + «СИНТЕТИЧЕСКОЕ МОТОРНОЕ МАСЛО» (кириллица) + «УНИВЕРСАЛЬНОЕ МАСЛО».  
- *Lacuna:* ActiPure (MCON_10) отсутствует.  
- *Contamination:* база фиксирует API SL / полусинтетику; 1L показывает **API SN/CF** + SYNTHETIC (locc_009_LACUNA) — требует верификации SKU.

**Вывод (K):** Макет 1L — **вертикальное сжатие** щитовой системы 4L без смены паттерна. Читаемость 5W-40 не страдает; страдают **вторичные клеймы** (ActiPure) и **маркер объёма** (1L подчинён 5W-40). Плотность выше, чем у Mobil 1L (нет «белого дыхания» вверху) — единый плотный стек от логотипа до футера.

| Контекст | 4L | 1L | Δ |
|----------|----|----|---|
| **Retail-полка** | Основная полка масел; полная замена | **Полка долива / top-up**; импульс, касса, аксессуары | **Смена M_CONTEXT** |
| **Плотность выкладки** | ~4–6 facings на п.м. | **~10–14 facings** (узкий профиль) | **↑ linear density** |
| **Дистанция считывания** | 1,5–3 м | 0,5–1,5 м (ближний контакт) | **Ближний контакт** |
| **E-commerce thumbnail** | LUXE + 5W-40 + золото — три якоря | LUXE + 5W-40 читаются; **1L, API SN/CF — ниже порога** (~120 px) | **Robustness ↓** для вторичных ТТХ |
| **M_NOISE** | `M_NOISE_VISUAL_HIGH` (база) | Тот же шум; **меньший footprint**, но **↑ альбедо** золотого корпуса | **Anomaly:** «золотой маяк» среди серебристых конкурентов |

**POV — System Dynamics:**  
- *Potential:* золотой корпус + высокая плотность facings → сильный salience в секции долива.  
- *Limit:* узкий силуэт теряется среди 4L-соседей на основной полке; 1L/API не проходят thumbnail-test.

**Вывод (M):** 1L живёт в **нише долива**, не конкурируя с 4L на основной полке. Золотой корпус даёт **высокий pop** в top-up-секции (контраст с серебром Mobil/Shell). E-commerce: primary layer (LUXE + 5W-40) проходит; объём и допуск — **нет**.

| Объём 4L → «полный цикл обслуживания» | **1L → «точечное пополнение»** | 🔄 **перекодировка MAP объёма** (locc_005) |

| Grip-dimples → контроль долива | **MAP не интегрирован** в K/S этикетки | ➕ LATENT (locc_001) |
| API SL (база) → «базовая защита» | API SN/CF на 1L → «современный допуск» | ⚠️ **Contamination / variant** — логический разрыв с базой 4L |
| Щит → безопасность (Gestalt-lock) | Щит **↑ % площади** | ✅ gestalt-completion **облегчена** |

**POV — Logical Coherence:**  
- *Conflict:* индустриальная «гипер-броня» (M_SYSTEM) vs portable top-up (C-геометрия) — **умеренный диссонанс**, как у Mobil, не блокирующий.  
- *Contradiction:* API SL (база) vs API SN/CF (1L) — **требует верификации** (разные SKU или ошибка базы).

**Вывод (MAP):** Ядро («золото = премиум», «5W-40 = телеметрия», «броня > жидкость») **масштабируется**. Искажаются: (1) карбон → соты, (2) семантика объёма, (3) ActiPure-MAP выпадает, (4) тактильный MAP не связан с визуальной системой. Щитовой gestalt на 1L **сильнее** за счёт доли площади.

### 2.6. EVAL — Оценка малого формата

| Метрика | 4L (база) | 1L (delta) | Δ |
|---------|-----------|------------|---|
| **Comprehensibility** | Оптимальная для brand + 5W-40 (база: 8/10 юзабилити) | **Высокая** для LUXE + 5W-40; **средняя** для API/1L | ↓ вторичный слой |
| **Cognitive load** | Высокая плотность, авторитарный стиль | **Равномерно высокая** по всей вертикали (нет «дыхания») | **↑ uniform density** vs Mobil bimodal |
| **Robustness (thumbnail)** | Высокая (LUXE + золото) | **Средне-высокая:** LUXE + 5W-40 ✅; API, 1L, соты — **fail** | ↓ вторичные |
| **Affective shift** | Снижение тревоги через «броню» (риторика 9/10) | Сохранён на близкой дистанции; золото **↑** на 0,5–1,5 м | context-dependent |
| **Structural fidelity** | Низкая (2/10) | Константа | — |
| **Rhetorical efficiency** | 9/10 (шумная полка) | **8/10** overall; **9/10** в top-up-контексте | золотой pop компенсирует сжатие |

**POV — Relational States:** compromise между **единым щитовым asset** (DRY) и потерей epistemic клеймов на 1L; parity M_SYSTEM при разной C-геометрии.

**Вывод (EVAL):** 1L **не ломает** primary value proposition (LUKOIL LUXE 5W-40). Ломается **secondary layer**: ActiPure, API-детализация, маркер 1L в thumbnail. Для ODSA: оценивать 1L и 4L **раздельными строками** — cognitive load на 1L **равномерно высокий**, не бимодальный.
