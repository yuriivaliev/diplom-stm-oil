## Паттерны рынка (сквозные)

### 5W-40 mass-mid
1. **Solid > Fluid** — броня/металл/термокинетика vs трибология.  
2. **5W-40 heuristic** — OEM/API off-front или microtype.  
3. **1L ≠ 4L** — secondary claims деградируют при DRY rescale.

### 0W-20 premium
4. **Hybrid gate** — institutional cue на фронте (HYSPEC > Mobil secondary badge).  
5. **API SP off-front** — white space для СТМ Modern/Tech.  
6. **Import / EN-facing** — RU lacuna у обоих tier-1; Mobil частично overlay.

### 10W-40 budget legacy (дополнение)
7. **Semi honesty + SAE front** — все три бренда; API **SG/CD front** (RU) vs **SN+ back** (Mobil).  
8. **SUPER hyperbole shared** — mesh / swirl / gold void при legacy chemistry.  
9. **1L doliv gap** — Mobil API off-front **усиливается** vs LUKOIL/GPN front SG/CD.
