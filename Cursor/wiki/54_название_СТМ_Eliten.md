## Message architecture (кратко)

### Master (все SKU)

> **Spec на этикетке — официально для РФ.**

### Sub-line

| Sub-line | SAE | Promise |
|----------|-----|---------|
| **Classic Protect** | 5W-40 syn · 10W-40 semi | Защита по readable spec, не по пафосу |
| **Modern Tech** | 5W-30 syn · 0W-20 FS | Инженерная spec на передней этикетке |

### SKU proof-lines

| SAE | Proof-line |
|-----|------------|
| **5W-40** | API SN/CF и OEM — на передней этикетке |
| **5W-30** | API SP · ACEA C3 — на фронте. OEM — на виду |
| **0W-20** | API SP · ACEA C5 · multi-OEM — на передней этикетке |
| **10W-40** | Полусинтетика · API на фронте · domestic OEM на виду |
