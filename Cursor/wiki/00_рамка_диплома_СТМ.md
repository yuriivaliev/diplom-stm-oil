## Тема

## География

- **Выводы диплома (продукт/SAE):** **ЦФО + СЗФО + ЮФО** = европейская часть РФ.
- **DR-A (бренды):** чаще **федеральный** контур + прокси по парку (AS-04).
- **Маркировка:** «Федеральный прокси — отсутствуют данные по ЦФО/СЗФО/ЮФО» — где нет ФО-ряда.
