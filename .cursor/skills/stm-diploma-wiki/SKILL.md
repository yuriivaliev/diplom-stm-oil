# STM diploma — LLM Wiki workflow

## Before any answer

1. Read `Cursor/AGENTS.md`.
2. Read `Cursor/wiki/index.md` → open **only** relevant pages (usually 2–3).
3. Check gate: `Cursor/wiki/08_декомпозиция_после_DR.md` — do not push stages 4–7 if 2a/3a open unless user asks.

## Answer rules

- Cite wiki pages and source IDs (S2-01, BX-02, V1-03…), not invented numbers.
- Mark **н/д**, **прокси**, **tier** where data is weak.
- **DR-A ≠ DR-B**; never mix S2 retail 2023 with V1 aftermarket 11M2023 in one table.
- Footnote URLs only from `Cursor/wiki/04_источники_и_URL.md` or `*_URL_реестр_*` — strip `utm_source=chatgpt.com`.
- Full DR reports: only via `Cursor/raw/README.md` when wiki is insufficient.

## After material changes

Update the matching `Cursor/wiki/XX_*.md`, `wiki/log.md`, and `wiki/overview.md` if the synthesis changed.
