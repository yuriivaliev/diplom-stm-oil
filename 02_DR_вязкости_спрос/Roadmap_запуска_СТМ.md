# Сводный чеклист (Roadmap) запуска СТМ моторных масел

**Горизонт:** 2026 год

## Фаза 1. Продуктовая архитектура и ТЗ (Завершено аналитически)
*   [x] **Утвердить бимодальную архитектуру бренда:** Разделение на линейку "Modern/Tech" (0W-20, 5W-30) и "Classic/Protect" (5W-40, 10W-40). *(Основание: CLA, Документ 6)*.
*   [x] **Утвердить матрицу запуска (Волна 1):** 5W-40 (Volume Driver) + 5W-30 (Core Balance). *(Основание: OST, Документ 9)*.
*   [x] **Утвердить матрицу запуска (Волна 2):** 0W-20 (Margin/Future Driver). *(Основание: OST, Документ 9)*.
*   [x] **Утвердить матрицу запуска (Волна 3):** 10W-40 (Tactical Coverage, ЮФО). *(Основание: OST, Документ 9)*.
*   [x] **Сформировать ТЗ для завода на 0W-20:** База Gr.III + PAO, пакет API SP/ILSAC GF-6A, защита от LSPI. *(Основание: RBS, Документ 10)*.
*   [x] **Сформировать ТЗ для завода на 5W-40:** База 100% Gr.III (HC), пакет API SN, Full SAPS, TBN > 10. *(Основание: RBS, Документ 11)*.
*   [x] **Закрыть лакуну по объему картера:** Установлено, что для целевых китайских авто (турбо 2.0л) сервисный объем заливки составляет 5.6 л (сухой 6.8 л).

## Фаза 2. R&D и Производство (Next Steps)
*   [ ] **Провести тендер среди блендеров (заводов):** Оценить возможности заводов в РФ (или за рубежом) по обеспечению ТЗ (особенно по доступу к ПАО и пакетам API SP для 0W-20).
*   [ ] **Разработать дизайн и утвердить фасовку упаковки:**
    *   **Линейка "Modern" (0W-20):** Премиальный дизайн, акцент на технологии. **Обязательная фасовка 5л + 1л** (в связи с сервисным объемом 5.6л).
    *   **Линейка "Classic" (5W-40):** Утилитарный дизайн, акцент на надежность, минимизация стоимости тары. Фасовка 4л + 1л.

## Фаза 3. Маркетинг и Дистрибуция (Next Steps)
*   [ ] **Разработать коммуникационную стратегию:**
    *   Для 0W-20: Месседж «Engineered for Asian Turbo» (API SP, ACEA C5).
    *   Для 5W-40/10W-40: Месседж «Maximum Engine Protection» (Защита от износа).
*   [ ] **Сформировать региональную стратегию продаж:**
    *   **ЦФО и СЗФО:** Фокус на продвижение 0W-20 и 5W-30 (СТО, маркетплейсы, специализированная розница).
    *   **ЮФО:** Фокус на push-продажи 10W-40 и 5W-40 (дистрибьюторы, рынки, традиционная розница).
*   [ ] **Определить ценовое позиционирование:**
    *   **0W-20:** Upper-Mainstream (на 10-15% ниже OEM-масел).
    *   **5W-40:** Mainstream/Economy (конкуренция с локальными лидерами по цене).
