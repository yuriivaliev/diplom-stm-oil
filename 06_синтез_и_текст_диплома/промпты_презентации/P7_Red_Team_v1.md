## Executive summary

| Критерий | v1 Oracle |
|----------|-----------|
| Анти-галлюцинации (DR, S2/V1, н/д) | ✅ Сильно |
| Анти-рекламность | ✅ Есть |
| Mockup / legal disclaimers | 🟡 Частично |
| Методология + нормативка на защите | 🔴 Не заданы явно |
| Timing 15 мин | 🟡 Нет бюджета времени |
| Narrative spine (пилот 5W-40) | 🔴 Не зафиксирован |
| Источники (ID S2-01…) | 🔴 Не обязаны на слайде |
| PGMM artifact vs canonical | 🔴 Отсутствует |

**Главный риск v1:** модель соберёт **логичную, но методологически «чистую на словах»** презентацию, которая на слайдах всё равно **смешает базы**, **перегрузит PGMM** или **пропустит нормативку/ограничения**, потому что prompt не задаёт **обязательные слайды**, **формат цифр** и **read order** проекта.

## Чеклист качества финальной презентации

### Достоверность
- [ ] Каждая цифра имеет ID источника (S2-01, BX-02…) или пометку **[н/д]**
- [ ] S2 и V1 **не** в одной таблице/графике
- [ ] DR-A (рынок) и DR-B (SAE) **не** смешаны в одном выводе
- [ ] Нет графика долей 2024–2026
- [ ] FIPS = регистрация ТЗ, не «legal clearance launch»
- [ ] PGMM: artifact vs canonical где релевантно

### Формат защиты
- [ ] 12 содержательных слайдов (+ титул опционально)
- [ ] Speaker notes суммарно ≤ 13 мин
- [ ] На deck ≤ 1 компактная таблица (≤5 строк)
- [ ] Полные матрицы только в приложениях
- [ ] Есть слайды: **методы**, **ограничения**, **нормативка (обзор)**, **результат/roadmap**

### Narrative
- [ ] Сквозная линия: рынок → SAE → white space → Eliten → mockups rev.2 → запуск
- [ ] Пилот **Classic Protect 5W-40 4L** назван явно
- [ ] DDx Verum/Calibr упомянут (не только победитель)
- [ ] Двойная рамка: диплом + проект СТМ

### Визуал
- [ ] Mockups подписаны: concept front · ART-001 · back TBD
- [ ] 1–2 mockup на слайде дизайна (не все 8)
- [ ] Нет рекламных superlative без доказательств

### Приложения
- [ ] А: матрица 5W-40 · Б: 0W-20 · Г: mockups rev.2 · Д: URL · FIPS скрины

## Вопросы комиссии (если deck по этому prompt)

1. **«Почему S2 2023, а не более свежие данные? Что с 2024–2026?»**  
   → Open data н/д; не интерполируем; retail S2 vs V1 раздельно.

2. **«Как вы отделили retail от aftermarket? Не смешали ли базы?»**  
   → S2 = retail 2023; V1 = aftermarket 11M2023; отдельные слайды/подписи.

3. **«На mockup указаны OEM/API — это уже подтверждённые спецификации?»**  
   → Concept front; TDS lock pending; back label regulatory layer TBD.

4. **«FIPS зарегистрирован — значит можно продавать?»**  
   → ТЗ кл. 04 = право на марку; маркировка CRPT, декларации, TDS — отдельный compliance track (гл. 7 / 06_F0b).

5. **«Почему 5W-40 пилот, а не 5W-30 как лидер SAE?»**  
   → DR-B: 5W-30 #1 по объёму; white space PGMM проработан на mass-mid 5W-40; пилот + Classic/Protect architecture.

6. **«Чем Eliten отличается от LUKOIL/GPN на полке — не копия?»**  
   → PGMM antidote (no armor/lattice); ODSA proof architecture; DDx от Verum/Calibr; engineering tone wiki/54.

7. **«Какие главные ограничения исследования вы признаёте?»**  
   → Федеральный прокси DR-B; н/д доли 2024–26; AI mockups; photo vs canonical; pre-launch без retail test @1.2m.
