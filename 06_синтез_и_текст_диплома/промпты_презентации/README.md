## Как запускать FINAL rev.2

1. Pre-read из FINAL (AGENTS, wiki, 04, 02–05, 08, 54, 55, 06_F0b).
2. Вставить в чат: **полный v2 oracle** + блок **P7-19…24** из FINAL.
3. Получить: карта материалов → deck-spec → приложения → риски → Q&A → self-check.
4. PPTX по deck-spec → финал: **`../презентация_защиты/ELITEN_defense_draft_v6.pptx`**.

**Новый визуал:** `../презентация_защиты/P7_Genspark_промпт_презентация.md` или `P7_Gamma_промпт_презентация.md` (2 этапа: тема → слайды).
