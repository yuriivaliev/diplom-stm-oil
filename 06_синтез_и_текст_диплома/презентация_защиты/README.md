## Workflow

### Защита (основной путь)
Работать в **`ELITEN_defense_Genspark_academic-review_final.pptx`** — правки Notes в PowerPoint, репетиция 14–15 мин.

После правок notes:
1. Обновить `SPEAKER_NOTES_CANON_Genspark_final.md` (или попросить agent)
2. Перегенерировать `ELITEN_defense_speaker_notes.docx`
3. Запись в `CHANGELOG_speaker_notes.md` и `журнал_проекта.md`

### Genspark (если новый export)
1. Этап 1 — шаблон → `согласовано: …`
2. Этап 2 — 17 слайдов + notes из CANON (не цифры 1–17)
3. Сверка с `SPEAKER_NOTES_CANON_Genspark_final.md`
